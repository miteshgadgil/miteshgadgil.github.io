Loan Granting Binary Classification

Microsoft Cortana Challenge

Can we predict whether a customer will be able to fully repay a loan?

PROBLEM CONTEXT

When a customer applies for a loan, banks and other credit providers use statistical models to determine whether or not to grant the loan based on the likelihood of the loan being repaid. The factors involved in determining this likelihood are complex, and extensive statistical analysis and modelling are required to predict the outcome for each individual case. I will implement a model that predicts loan repayment or default based on the data provided.

DATA

The dataset used in this competition consists of synthetic data that was generated specifically for use in this project. The data is designed to exhibit similar characteristics to genuine loan data. The dataset consists of over 255,000 loan records and 19 columns(attributes). The dataset can be downloaded as a csv file by clicking here and a description of the attributes of the dataset can be obtained here.

MODELLING RESULTS

Having created a cleaned dataframe and binding the engineered features I used the Random Forest Classifier model from the scikit-learn module in python. I tuned its parameters using gridsearch and obtained an estimate of the performance of the model using the 'Out-of-box' error for the cross-validations. The oob score (substitute for CV score) was 0.83.

Next I used the fitted model, to predict on the test set which was held out during the training process. The model performed pretty well with an accuracy of 84.8%. In an unbalanced dataset, the F-score is a better metric to optimise compared to the accuracy. The F-score for the predictions was 0.39, which can be bettered by using grid search to optimize on f-score rather than oob score, which is the default.
