MY PROJECTS

A complete list of all my projects
from research to hobby - Everything's here!!


NOTE:
Most of the projects are on Github (it's open sourced). If a project interests you, please do visit the project's webpage.

More updates on the way, thank you for your time :)


  1. An IoT-Based Energy Management System for Data Centers

    This is an IoT solution that allows administrators to monitor the energy consumption at Datacenters. The sensor nodes sense: Humidity, Temperature and the Voltage consumed in each rack. The data acquired gets processed and prediction of future needs is done with primitive perceptron learning algorithm[More..]

  2. The Smart Bus Project : Redefining Commuting in Cities

    This project was part of my undergraduate thesis. The project implements an end-to-end system that establishes communication between the bus and the commuter. A project done under the “Smart Cities” context, this project predicts ETA and also helps commuters track their ride. The project also has a admin console to monitor the fleet of buses.[More..]

  3. An Automated System to Mitigate Loss of Life at Unmanned Level Crossings

    A prototype of an early warning system that signals about an incoming train at unmanned railway level crossings. This project was done as part of the Texas Instruments Innovation Challenge India Design Contest 2015, with kits sponsored by Texas Instruments, India[More..]

  4. Super Mario for Web Browsers

    A HTML/Javascript reproduction of the famous arcade game "Super Mario", built as part of the Free and Open Source Software course at Amrita Vishwa Vidyapeetham. Built using Grunt, HTML5, JS - this game is supported on the common web browsers[More..]
